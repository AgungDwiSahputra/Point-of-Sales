# Point-of-Sales

Static POS System (Jamstack) — Astro, Tailwind CSS, Preact, Supabase, Dexie.js.

Lihat spesifikasi lengkap di [docs/PRD_POS_Static_Jamstack_v2.md](docs/PRD_POS_Static_Jamstack_v2.md).

## Commands

| Command           | Action                                       |
| :----------------- | :-------------------------------------------- |
| `npm install --legacy-peer-deps` | Install dependencies (`--legacy-peer-deps` wajib - `@vite-pwa/astro` belum menyatakan dukungan resmi utk Astro 7 di peer deps-nya, walau berfungsi normal) |
| `npm run dev`       | Start local dev server at `localhost:4321`    |
| `npm run build`     | Build production site to `./dist/`            |
| `npm run preview`   | Preview build locally                         |
